About Flood Prediction System

An AI-powered web application that predicts flood risk using Machine Learning and historical rainfall data from India.

Project Overview

This application predicts flood risk based on monthly and seasonal rainfall values using Machine Learning algorithms. It helps users estimate flood possibility from rainfall patterns.

Dataset

Rainfall in India (1901–2015)

• 4116 Rainfall Records
• Historical Rainfall Dataset
• Missing Values handled using Mean Imputation
